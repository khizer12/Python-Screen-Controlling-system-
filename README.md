# Python-Screen-Controlling-system-

Minimal Requirements

Minimum System Requirements:

OS: Windows 10, macOS 10.14+, or Ubuntu 18.04+
Python: Python 3.8 or newer
RAM: 4GB minimum, 8GB recommended
Network: Stable LAN connection (Wi-Fi or Ethernet)

Recommended System Requirements:

OS: Windows 11, macOS 12+, or Ubuntu 20.04+
Python: Python 3.9+
RAM: 8GB or more
Network: Gigabit Ethernet for best performance
CPU: Multi-core processor (Intel i5/Ryzen 5 or better)
GPU: Dedicated GPU for hardware decoding

For most users, this single command should work:

pip install av opencv-python pillow pynput

Then run the client with:

python client.py
